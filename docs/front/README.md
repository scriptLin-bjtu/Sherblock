## 技术栈

### 后端
- **WebSocket服务器**：`ws`
- **文件监听**：`chokidar`
- **HTTP服务器**：`express`

### 前端
- **语言**：原生 JavaScript (ES6+)
- **构建工具**：Vite
- **UI**：原生HTML+CSS
- **Markdown渲染**：`marked` 库

## 预计开发周期

**4-6周**，分6个阶段逐步实现。
