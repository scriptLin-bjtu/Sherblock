# 项目概览

## 一、项目背景

### 当前状态
- **运行方式**：纯终端交互式应用
- **入口**：`src/index.js`，通过 `npm start` 启动
- **交互方式**：使用 `readline` 进行用户输入/输出
- **数据存储**：`data/` 目录下按工作区隔离存储

### 改造目标
将现有的终端应用改造为具有WebSocket服务器的Web应用，提供Claude风格的可视化界面：
- **左侧**：工作区列表（显示data目录下的工作区列表）
- **中间**：对话区域（Agent执行过程和对话消息流）
- **右侧**：工作区详情（显示scope、charts、reports、logs等）
- **中间下方**：输入框（用于与Agent交互）
