## 技术栈选择

### 后端
- **WebSocket服务器**：`ws` 或 `socket.io`
- **文件监听**：`chokidar`（监听data目录变化）
- **HTTP服务器**：`express`（提供静态文件服务）
- **现有技术**：保持现有的Agent系统不变

### 前端
- **语言**：原生 JavaScript (ES6+)
- **构建工具**：Vite
- **UI组件**：原生HTML+CSS，模块化JavaScript
- **WebSocket客户端**：原生WebSocket API
- **Markdown渲染**：`marked` 库
- **图表显示**：直接嵌入SVG
